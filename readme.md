# Data Structures & Algorithms

A personal journal of my journey through Data Structures and Algorithms in C++.
This repository tracks practice problems, pattern exercises, and notes I've
written while building a strong foundation in DSA.

---

## About

This repo is a structured record of consistent practice — each folder maps to a
topic I'm working through, and each file is a self-contained solution. The goal
is not just to solve problems, but to internalize the underlying patterns and
problem-solving techniques used in competitive programming and technical
interviews.

**Language:** C++
**Status:** Actively maintained

---

## Repository Structure

```
DATA-STRUCTURES-ALGORITHMS/
├── basic/            # Warm-up problems (odd/even, swaps, sums, etc.)
├── basic-maths/      # Mathematical problems (digits, palindromes, reverse)
├── patterns/         # Pattern printing — logic-building exercises (1–19)
├── strings/          # String manipulation theory and problems
├── THEORY/           # Concept notes and reference material
├── random-learning/  # Experimental code and scratchpad
├── template.cc       # Base C++ template for quick problem setup
└── advacned_template.cc
```

## Getting Started

### Prerequisites
- A C++ compiler (`g++` recommended, C++17 or later)
- Any code editor (VS Code config is included)

### Compile & Run

```bash

g++ patterns/first.cpp -o first

# Run the executable
./first
```
